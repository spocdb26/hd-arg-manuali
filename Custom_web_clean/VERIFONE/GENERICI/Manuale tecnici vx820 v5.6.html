1. Scopo e campo di applicazione

Questo documento descrive le modalità di utilizzo e configurazione del pos VX820.

2. Interfacciamento con il sw in cassa

Questo dispositivo richiede l'interfacciamento con il software di cassa, che può avvenire in modalità

seriale o TCP/IP.

3. Accensione

Accensione : collegare l'alimentatore del pos ed attendere qualche secondo.

Un esempio della schermata iniziale è mostrata sotto.

4. Configurazione

4.1.

Parte Pos: Configurazione SSA manager

F1 permette di entrare nel MENU' di MULTIGT application.

Per la configurazione:

1. F1 (sul display in alto a sx ) e digitare la password: 0461;

2. Selezionare CONFIGURAZIONE;

3. ATTIVO AMONEY → SI';

4. RESET TER → NO;

5. INSERIRE RUPP → inserire RUPP da tastiera (i tasti sono alfanumerici) → VERDE;

6. DISAB SOVR → SI';

7. DEBUG → NO;

8. INVIO DLL Scontrino → scegliere SI’

9. SALVARE → SI';

Configurare la linea principale:

1. F1 e digitare la password: 0461;

2. selezionare LINEA SSA;

1. TIPO LINEA → ETH;

2. CONNESSIONE → TCPIP;

3. IP → inserire IP verso SSA con tastiera poi premere tasto VERDE (per inserire il . premere

F1);

4. PORTA → inserire PORTA di SSA con tastiera poi premere tasto VERDE;

5. ATTESA SOCKET → inserire 05 poi premere tasto VERDE;

6. ATTESA RISPOSTA → inserire 05 poi premere tasto VERDE;

7. SALVARE CONFIG → SI'

NOTE: i parametri, dopo la prima volta che sono stati inseriti, rimangono salvati, se si vogliono

mantenere, basterà confermarli con il tasto VERDE, altrimenti premere il tasto GIALLO e

inserire quello desiderato.

Configurare la linea backup:

1. F1 e digitare la password: 0461;

2. selezionare LINEA SSA BACKUP;

1.

TIPO LINEA → ETH;

2.

CONNESSIONE → TCPIP;

3.

IP → inserire IP di backup verso SSA con tastiera poi premere tasto VERDE (per inserire il .

premere F1);

4. PORTA → inserire PORTA di SSA di backup con tastiera poi premere tasto VERDE;

5. ATTESA SOCKET → inserire 05 poi premere tasto VERDE;

6. ATTESA RISPOSTA → inserire 05 poi premere tasto VERDE;

7. SALVARE CONFIG → SI'

Configurare l'IP del POS

1. F1 e digitare la password: 0461;

2. selezionare IP POS;

1. USO DHCP? NO

2. IP POS → inserire indirizzo IP assegnato al POS

3. SUBNET MASK → inserire subnet mask (normalmente 255.255.255.0)

4. DEFAULT GATWAY → inserire default gatway

5. DNS → lasciare vuoto

6. SALVARE CONFIG → SI'

NOTE: i parametri, dopo la prima volta che sono stati inseriti, rimangono salvati, se si vogliono

mantenere, basterà confermarli con il tasto VERDE, altrimenti premere il tasto GIALLO e

inserire quello desiderato.

4.2.

Attivazione del pos

Per far partire il primo DLL:

1. F1 e inserire password 0461

2. selezionare ATTIVAZIONE (premere le frecce in basso a destra sul pos per scorrere le varie

opzioni della tabella)

NOTA: una volta che è stato eseguita il dll, controllare in AMONEY che nella colonna OpDaEseguire la

lettera A sia diventata N.

Questo significa che l'attivazione è andata a buon fine.

5. Configurazione BPE

5.1.

Configurazione

1. Selezionare F4;

2. Selezionare BUONI PASTO;

3. Selezionare CONFIGURAZIONE:

1. PORT SERVER BP → inserire la PORTA 2689, poi premere tasto VERDE;

2. PORT SERVER BACKUP BP → inserire PORTA di backup del server dedicato, poi

premere tasto VERDE;

3. TIMEOUT → inserire 120, poi premere tasto VERDE (per inserire il . premere F1);

4. ECCEDI IMPORTO → selezionare OFF, poi premere tasto VERDE;

5. PROTOCOLLO ECR V2 → OFF poi premere tasto VERDE.

Selezionare GESTIONE e inserire password 72618;

3. Selezionare DOWNLOAD INI e confermare col tasto

4.

6.2.

Test chiamata

1.

Selezionare F4;

2.

Selezionare GESTIONE e inserire password 72618;

3. selezionare la voce TEST CHIAMATA

Il POS verifica che le connessioni principale e di backup funzionino correttamente con il server ssa; dopo
il test sul display del POS appariranno i risultati. (OK per la connessione funzionante, KO altrimenti).

8. Reset del POS

8.1.

Reset totale del pos

Nel caso in cui risulti necessario cancellare tutti i dati caricati a bordo del pos (relativi a tutti i terminalID
attivati a bordo), eseguire le seguenti operazioni:

1. F1 e digitare la password: 0461;

2. Selezionare CONFIGURAZIONE;

3. ATTIVO AMONEY → SI'

4. RESET TER → SI'

5. CONFERMI RESET → SI'

6. CHIUSURA TERMID → SI'

8.2.

Reset dei dati di una sola banca

Il reset del POS viene utilizzato per cancellare i dati del termID e inserirne uno nuovo.

Per resettare il terminale:

1. selezionare Menu (in basso a destra sul display del pos);

2. selezionare INSTALLATORE;

3. inserire password 183729;

4. selezionare RESET TERMINALE;

5. Eliminare config? CONFERMI → SI';

Per rifare l'attivazione vedere i paragrafi 4.2.

9. Aggiornamento POS

Il pos impiegherà circa una decina di minuti ad eseguire l’aggiornamento, al termine va verificato che sia
presente sul pos la release corretta.

1. F1 e inserire password 0461

2. scendere fino alla voce TMS e selezionarla

3. inserire l’IP SSA, se già inserito verificare che sia corretto

4. inserire porta 2667, se già inserita verificare che sia corretta

5. inserire il TMS ID 00000200

6. salvare e lanciare → SI

Il pos si collegherà al nostro server ed effettuerà lo scarico della release. Al termine effettuerà il riavvio.
Non effettuare nessuna operazione finché il terminale non tornerà alla schermata iniziale con il logo.

Verifica della release:

CB2: selezionare il tasto rel e verificare che compaia CB203.69 REV_D, poi premere tasto rosso finché
non torna nella schermata principale con il logo

MGTAPP: selezionare F1 e inserire password 0461, scendere e selezionare la voce INFO AMONEY e

verificare che la versione sia 1.16 (per uscire tasto rosso)

ARGECR: selezionare F4, scendere e selezionare la voce About ARGECR e verificare che la versione sia

ARGECR_3.00J (per uscire tasto rosso)
