1. Scopo e campo di applicazione

Questo documento descrive le modalità di utilizzo e configurazione del pos VX820.

2. Interfacciamento con il sw in cassa

Questo dispositivo richiede l'interfacciamento con il software di cassa, che può avvenire in modalità

seriale o TCP/IP.

3. Accensione

Accensione : collegare l'alimentatore del pos ed attendere qualche secondo.

Un esempio della schermata iniziale è mostrata sotto.

4. Configurazione

4.1.

Parte Pos:Configurazione SSA manager

F1 permette di entrare nel MENU' di MULTIGT application.

Per la configurazione:

1. F1 (sul display in alto a sx ) e digitare la password: 0461;

2.

Selezionare CONFIGURAZIONE;

3. ATTIVO AMONEY → SI';

4. RESET TER → NO;

5. INSERIRE RUPP → inserire RUPP da tastiera (i tasti sono alfanumerici) → VERDE;

6. DISAB SOVR → SI';

7. DEBUG → NO;

8. SALVARE → SI';

Configurare la linea principale:

1. F1 e digitare la password: 0461;

2. selezionare LINEA SSA;

1. CONNESSIONE → TCPIP;

2. IP → inserire IP verso SSA con tastiera poi premere tasto VERDE (per inserire il . premere

F1);

3. PORTA → inserire PORTA di SSA con tastiera poi premere tasto VERDE;

4. ATTESA SOCKET → inserire 05 poi premere tasto VERDE;

5. ATTESA RISPOSTA → inserire 05 poi premere tasto VERDE;

6. SALVARE CONFIG → SI'

NOTE: i parametri, dopo la prima volta che sono stati inseriti, rimangono salvati, se si vogliono

mantenere, basterà confermarli con il tasto VERDE, altrimenti premere il tasto GIALLO e

inserire quello desiderato.

Configurare la linea backup:

1. F1 e digitare la password: 0461;

2. selezionare LINEA SSA BACKUP;

1.

CONNESSIONE → TCPIP;

2.

IP → inserire IP di backup verso SSA con tastiera poi premere tasto VERDE (per inserire il .

premere F1);

3. PORTA → inserire PORTA di SSA di backup con tastiera poi premere tasto VERDE;

4. ATTESA SOCKET → inserire 05 poi premere tasto VERDE;

5. ATTESA RISPOSTA → inserire 05 poi premere tasto VERDE;

6. SALVARE CONFIG → SI'

Configurare l'IP del POS

1. F1 e digitare la password: 0461;

2. selezionare IP POS;

1. USO DHCP? NO

Se il POS utilizza un IP statico:

1.3 IP POS → inserire indirizzo IP assegnato al POS

1.4 SUBNET MASK → inserire subnet mask (normalmente 255.255.255.0)

1.5 DEFAULT GATWAY → inserire default gatway

1.6 DNS → lasciare vuoto

1.7 SALVARE CONFIG → SI'

NOTE: i parametri, dopo la prima volta che sono stati inseriti, rimangono salvati, se si vogliono

mantenere, basterà confermarli con il tasto VERDE, altrimenti premere il tasto GIALLO e

inserire quello desiderato.

6. Aggiornamento POS

Il pos impiegherà circa una decina di minuti ad eseguire l’aggiornamento, al termine va verificato che sia
presente sul pos la release corretta.

  1. F1 e inserire password 0461
  2. scendere fino alla voce TMS e selezionarla
  3. inserire l’IP SSA, se già inserito verificare che sia corretto
  4. inserire porta 2667, se già inserita verificare che sia corretta
  5. inserire il TMS ID 00000200
  6. salvare e lanciare → SI

Il pos si collegherà al nostro server ed effettuerà lo scarico della release. Al termine effettuerà il riavvio.
Non effettuare nessuna operazione finché il terminale non tornerà alla schermata iniziale con il logo.

7. BPE-Configurazione

Per accedere al menù di configurazione di AmoneyBPE:

  • premere sul pos F4
  • selezionare 2-Buoni Pasto
  • selezionare 1-Configurazione

Nel menù Configurazione si trovano i parametri di configurazione per Pagamento BPE:

• Port server BP: inserire la porta (per eventualmente cancellarla/modificarla tasto giallo) indicata
nelle note su Amoney.
Premere tasto verde per salvare e passare alla voce successiva.

• Timeout seconds: Il timeout è rappresentato in secondi, va impostato a 120

• Eccedi importo: la voce va impostata off = disabilita scegliere sempre questa opzione

• Max num BPE: impostare a 99.
