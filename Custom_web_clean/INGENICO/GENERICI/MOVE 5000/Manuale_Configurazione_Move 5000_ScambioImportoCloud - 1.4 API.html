1.1.1. Configurazione SSA manager

Per accedere al menù di configurazione dell’SSA manager è necessario:

1. premere tasto ”F” sul tastierino in basso a sinistra

2. selezionare la voce SSA MANAGER,

3. selezionare la voce CONFIGURAZIONE e digitare la password 2010

e procedere con gli step sotto descritti.

1. Selezionare CONFIG SERVER/RUPP:

SLOT: selezionare ARGENTEA e premere tasto verde

MERCHANT SERVER: ARG-!-NTSV.AMON e premere tasto verde

RUPP: digitare il RUPP assegnato e premere tasto verde per confermare

il POS ritorna alla voce SLOT, premere tasto rosso e confermare con tasto verde il salvataggio

delle modifiche.

2. Selezionare LINEA SSA PRINCIPALE:

  1. TIPO LINEA: selezionare GPRS e confermare con tasto verde
  2. TIPO PROTOCOLLO: selezionare TCP/IP e confermare con tasto verde
  3. INDIRIZZO IP: 208.224.253.5
  4. e confermare con tasto verde
  5. PORTA: 3 4032
  6. e confermare con tasto verde
  7. TIMEOUT SOCKET: 10 e confermare con tasto verde
  8. TIMEOUT RICEZIONE: 10 e confermare con tasto verde
  9. il POS ritorna alla voce TIPO LINEA, premere tasto rosso e confermare con tasto verde il

salvataggio delle modifiche.

3. Selezionare LINEA SSA BACKUP:

  1. TIPO LINEA: selezionare GPRS e confermare con tasto verde
  2. TIPO PROTOCOLLO: selezionare TCP/IP e confermare con tasto verde

salvataggio delle modifiche.

5. Configurazione CB2 (Per Scambio Importo Cloud)

Per accedere al menù di configurazione generale del POS è necessario:

1. premere il pulsante F1,

2. selezionare la voce “Installazione”,

3. digitare la password 0107 e procedere con gli step sotto descritti.

Selezionare Configura/DatiLinea ECR e impostare i dati come segue:

  1. Protocollo = 25
  2. Tipo Linea = Ethernet
  3. Scambio Imp Obbligatorio = SI
  4. Disabilita TRS da POS = SI

Premere tasto rosso e poi tasto verde per salvare la configurazione.

6. Configurazione Applicazione ArgPayMgr

Per accedere al menù di configurazione dell’applicazione è necessario:

1. premere tasto ”F” sul tastierino in basso a sinistra

2. selezionare l’icona ArgPayMng

e procedere con gli step sotto descritti.

Selezionare la voce “Configurazione” (password 72619) e configurare le seguenti voci come segue,

premendo tasto verde per confermare ciascuna scelta:

Parametro

Valore

RUPP

Codice di 16 cifre DI SOLITO INIZIA PER “N”

Server IP

208.224.253.5

Server Port

3 4032

Server IP bkp

208.224.253.5

Server Port bkp

34032

Server timeout

30 (secondi)

Connessione

GPRS

CUSTOM

APN: m2m.tns.prod1

Scelta applicazione

RICHIESTA IMPORTO SSA

Max num notifiche offline

Selezionare il numero di notifiche che si vogliono salvare

qualora il pos non possa connettersi al server SSA, tali

notifiche saranno inviate alla successiva connessione

Max num invii notifiche

Selezionare il numero di notifiche che si vogliono inviare

dopo il primo pagamento in cui il pos ristabilisce la

connessione al server SSA

NB.: Una volta raggiunto il numero massimo di notifiche salvate in memoria, alla successiva notifica che è
necessario salvare a causa della mancata connessione col server SSA, viene rimossa la notifica più vecchia
in memoria per fare spazio alla più recente.

7. Configurazione Signa

  1. premere tasto Grigio (sopra il tasto rosso)
  2. selezionare SIGNA - psw 7446;
  3. per la configurazione:
  4. selezionare MENU SIGNA
  5. selezionare PARAM.LINEA TML - psw 7454
  6. selezionare PARAMETRI LINEA 1; i parametri di configurazione sono
  7. tipo linea: selezionare GPRS e premere tasto verde;
  8. tipo protocollo: selezionare TCP/IP + SSL e premere tasto verde;
  9. ID. CERT.SSL3 : digitare 27 e premere tasto verde
  10. NUM TELEFONICO/ IP: digitare IP SSA principale 208.224.253.5 e premere tasto verde;
  11. WEB SERVICE: inserire /idrm_argentea/wsincoming/wsincoming.aspx e premere tasto verde;
  12. PORTA: digitare 34054 e premere tasto verde;
  13. apn gprs: m2m.tns.prod1
  14. username lasciare vuoti
  15. pw lasciare vuoti
  16. ritorna in tipo linea, premere tasto rosso e confermare con tasto verde le
  17. modifiche.
  18. A questo punto SIGNA e configurato ed le firme vengono inviate in maniera corretta.
