2. Attivazione bancaria.

2.1.

Parte Pos: configurazione SSA manager

Per accedere al menù SSA manager e procedere con la

configurazione per l’attivazione:

1. premere tasto Menu

(posto in alto a dx)

  1. selezionare SSAMANAGER
  2. selezionare CONFIGURAZIONE – psw 2010.
  3. selezionare Config server rupp:

• Slot → selezionare lo slot Argentea ETH per lo scambio

importo tramite tcp/ip;

• Merchant server: compare ARG-!-NTSV.AMON Premere tasto verde per confermare;

  • RUPP: inserire il rupp assegnato, per la lettera iniziale utilizzare la tastiera visualizzata a
  • video, per i valori numerici utilizzare la tastiera numerica del pos. Premere tasto verde
  • per continuare.
  • Quando il pos ritorna in slot, premere tasto rosso, dopo di che, verrà richiesto se si
  • desidera salvare le modifiche, premere tasto verde per confermare.

5. Selezionare Linea SSA Principale:

  • selezionare Tipo Linea: scegliere tra Ethernet e confermare con tasto verde;
  • selezionare Tipo Protocollo: scegliere tra TCP/IP e Confermare con tasto verde;
  • inserire indirizzo IP: 192.168.10.220 (valido per Alice Pizza diretti) confermare con
  • tasto verde;
  • inserire Porta: 2608. Confermare con tasto verde;
  • inserire Timeout socket: Confermare con tasto verde;
  • inserire Timeout Ricezione: Confermare con tasto verde;

3. Verifica SIGNA

Per l’esercente Alice Pizza non è al momento contemplato il servizio SIGNA. Durante

l’installazione, qualora la voce SIGNA fosse presente nel menù principale, occorre verificare che

sia disabilitata.

1. premere tasto Grigio (sopra il tasto rosso)

  1. selezionare SIGNA - psw 7446;
  2. per la configurazione:
  • selezionare MENU.SIGNA
  • selezionare PARAM. FIRMA
  • selezionare ABILTAZIONE FIRMA SI/NO –
  • IMPOSTARE SU NO
  • premere tasto rosso e confermare con tasto verde per salvare le modifiche

5. Aggiornamento terminale

5.1.

AGGIORNAMENTO ULTIME RELEASE DISPONIBILI

  1. Premere il tasto F1;
  2. Selezionare la voce SERVIZIO e poi Tasto VERDE;
  3. Selezionare la voce TELEGESTIONE e poi Tasto VERDE;
  4. Inserire la password 4378 e poi Tasto VERDE;
  5. Selezionare la voce CONFIGURA LINEA e poi Tasto VERDE;
  6. TIPO LINEA - ETHERNET e selezionarla col tasto VERDE;
  7. Passare alla voce successiva premendo nuovamente Tasto VERDE.
  8. PROTOCOLLO DI RETE - TCP/IP e selezionarla col tasto VERDE;
  9. Passare alla voce successiva premendo nuovamente Tasto VERDE.
  10. PROT. DI TRASPORTO - lasciare la voce che trovate e andare avanti con Tasto VERDE.
  11. INDIRIZZO IP/URL- digitare 192.168.10.220 e premere Tasto VERDE.
  12. PORTA: digitare la porta 5001 e premere Tasto Verde.
  13. Premere ora tasto ROSSO, e poi il Verde per confermare il salvataggio delle

configurazioni;

Una volta configurato il POS si puo avviare la telegestione in questo modo:

  1. Premere il tasto F1;
  2. Selezionare la voce SERVIZIO e poi Tasto VERDE;

6. Configurazione scambio importo

6.1.

CONFIGURAZIONE MANUALE ARGPAYMAGR

  • 1.premere Menu (posto in alto a dx)
  • Selezionare la voce ArgPayMgr
  • Selezionare la voce Configurazione
  • Digitare la password 72619
  • Nella voce di menu Configurazione impostare i seguenti parametri:
  • Rupp: verificare che il valore presente corrisponda al RUPP fornito nel file di installazione
  • Server IP: 192.168.10.220
  • Server Port: 2608
  • Server IP Backup: 192.168.10.221 PORT 2690
  • Server Timeout: lasciare valore di default
  • Connessione: selezionare Ethernet
  • Scelta applicazione: SELEZIONARE ECR
  • Max num notifiche offline: lasciare il valore di default
  • ECR PORT: 40001

Premere tasto giallo per salvare le modifiche.

E’ possibile che il pos esegua un riavvio.

7. Configurazione AmoneyBPE + Pagamenti ADV

Per accedere al menù di configurazione dell’applicazione AmoneyBpe:

  1. premere sul POS il tasto grigio
  2. selezionare la voce AMONEYBPE
  3. selezionare la voce Configurazione
  4. digitare la password 72619

Nel menù Configurazione è possibile impostare i seguenti parametri per la gestione dei BPE:

• RUPP: inserire il rupp corrispondente (per cancellare/modificare tasto giallo).

Premere tasto verde per salvare.

  • Tipo Connessione: Ethernet
  • Protocollo di Rete: TCP/IP
  • Conn Principale: verrà richiesto di inserire:
  • IP server BP: 192.168.10.220 premere tasto verde per salvare e passare alla voce
  • successiva
  • Port server BP: 2609 premere tasto verde per salvare.
  • Conn Backup: verrà richiesto di inserire:
  • IP server BP: 192.168.10.221 premere tasto verde per salvare e passare alla voce
  • successiva.
  • Port server BP: 2691 premere tasto verde per salvare.
  • Porta ECR: se non è già inserita, inserire il valore 02571 sempre.
  • Premere tasto verde per salvare.
  • Time-out: Si consiglia di impostarlo a 120, è rappresentato in secondi.
  • E’ il tempo che il POS aspetta per ricevere/inviare messaggi al server.
  • Premere tasto verde per salvare.
  • Eccedi importo: si consiglia di impostarlo su OFF.
  • Standalone: NO
  • Premere tasto verde per confermare.
  • Demat seq code: lasciare il parametro impostato di default
  • Premere tasto verde per confermare.
  • Protocol ECR V2: ON
  • Premere tasto verde per salvare e passare alla voce successiva.
  • Opzioni Stampa: ECR tutti gli scontrini verranno stampati dalla cassa
  • Porta SSA: 2608
  • Premere tasto verde per salvare.

Terminata la configurazione premere il tasto giallo, i dati si salveranno automaticamente.
