1. Scopo e campo di applicazione

Questo documento descrive le modalità di utilizzo e configurazione del pos Ingenico Lane 3000 per i PV

Brendolan - Maxidi

2. Accensione e Riavvio

2.1.

Accensione

Accensione : collegare l'alimentatore del pos. Il pos si troverà nella schermata di idle dopo circa 90
secondi.

2.2.

Riavvio

Riavvio: premere contemporaneamente il tasto giallo e il tasto punto.

3. SIGNA

Il pos Ingenico Lane 3000 permette di apporre la firma del pagamento direttamente a display del pos. La
firma va poi inviata al server SIGNA, per questo, una volta eseguita l’attivazione, va configurato il menù
SIGNA.

Per i punti vendita Brendolan – MaxiDi il servizio SIGNA VA DISATTIVATO.

3.1.

Disattivazione Signa

Per accedere al menù SIGNA e procedere con la configurazione:

1.

premere tasto F (sopra il tasto rosso)

2.

selezionare SIGNA - psw 7446;

3. per la configurazione:

  • selezionare MENU SIGNA
  • selezionare PARAM. FIRMA
  • selezionare NO

3.2.

Cancellazione firme

1.

premere tasto F (sopra il tasto rosso)

2.

selezionare SIGNA - psw 7446;

3. selezionare MENU SIGNA

4. selezionare RESET FILE FIRME – PW 62883607423116

5. conferma RESET – tasto verde

4. Attivazione bancaria.

salvare le modifiche, premere tasto verde per confermare.

5. Configurazione scambio importo

5.1.

CONFIGURAZIONE MANUALE

Lo scambio importo dipende dallo slot che si sceglie nel menù SSA MANAGER:

• scegliere Argentea ETH per lo scambio importo tramite ETHERNET

NB!!! VERIFICARE SEMPRE CHE LE IMPOSTAZIONI DI SCAMBIO IMPORTO SIANO STATE IMPOSTATE

CORRETTAMENTE

Per modificare a mano le impostazione dello scambio importo:

premere tasto F1;

  1. selezionare Installatore (0107);
  2. selezionare Configura;
  3. selezionare Dati Linea ECR

4. CONFIGURAZIONE PROTOCOLLO ETHERNET ARGENTEA 25

  1. protocollo: 25
  2. tipo linea: modalità di scambio importo ETH;
  3. porta: 40001
  4. Scambio Imp. Obbl: impostare su SI;
  5. Stampa ticket: impostare su ECR;
  6. Disabilita trs. Da pos: impostare su SI;

5.2.

Reset configurazione scambio importo

Per eliminare completamente la configurazione dello scambio importo o per riportare i valori di default
dello slot eseguire:

  1. selezionare Tasto F;
  2. selezionare SSA MANAGER – CONFIGURAZIONE ( psw 2010)
  3. selezionare CONFIGURAZIONE ECR, questa permette di eliminare/riconfigurare le impostazioni per

lo scambio importo:

◦ Se si seleziona disabilita e si preme tasto verde, il pos si riavvierà, al termine del quale non sarà

più “attivo” lo scambio importo.
◦ Se si seleziona abilita, il pos si riavvierà e imposterà lo scambio importo con i valori di default

dati segnalati dallo slot

6. Reset Pos

6.1.

Reset totale

Sul pos TETRA è possibile resettare i termid e i parametri di configurazione dell’ssa oppure solamente i
termid.
Per resettare completamente il pos:

  1. premere TASTO F
  2. selezionare SSA MANAGER – CONFIGURAZIONE (2010)
  3. selezionare RESET PARAMETRI

• a display verrà richiesto di eseguire il reset completo: se si seleziona OK-tasto verde verrà eseguita
una cancellazione completa, cioè verranno eliminati sia i termid presenti a bordo sul pos che i
parametri di configurazione dell’ssa manager.

Per riconfigurare e attivare il pos vedere capitolo 4.

6.2.

Reset bancario

Per resettare solamente i termid e non i parametri di configurazione:

  1. premere TASTO F
  2. selezionare SSA MANAGER – CONFIGURAZIONE (2010)
  3. selezionare RESET PARAMETRI

• Se alla richiesta di cancellazione parametri completa si preme tasto rosso il pos chiederà di

eseguire la cancellazione bancaria. A questo punto premendo tasto verde viene eseguita un reset

parziale e cioè verranno eliminati solamente i termid presenti sul pos, mentre verranno lasciati

invariati i parametri di configurazione.

7. Configurazione BPE

7.1.

Configurazione Amoney BPE

Per accedere al menù SSA manager e procedere con la configurazione per l’attivazione:

  • 1.
  • premere tasto F/GRIGIO
  • selezionare AMONEYBPE
  • selezionare CONFIGURAZIONE – psw 72619
  • RUPP: inserire il RUPP corrispondente (per cancellare modificare tasto giallo). Premere
  • il tasto verde per salvare
  • Tipo Connessione: Ethernet tasto verde per salvare e passare alla voce successiva
  • Protocollo di rete: TCP/IP tasto verde per salvare e passare alla voce successiva
  • Conn Principale:

 IP server BP: 10.240.0.1 tasto verde per salvare e passare alla voce successiva

 Port server BP: 2685 tasto verde per salvare e passare alla voce successiva

• Conn Bakcup:

  •  IP server BP: 10.240.0.9 tasto verde per salvare e passare alla voce successiva
  •  Port server BP: 2691 tasto verde per salvare e passare alla voce successiva
  • Porta ECR: se non già inserita, inserire 02571 tasto verde per salvare e passare alla voce
  • successiva
  • Timeout: 120 tasto verde per salvare e passare alla voce successiva
  • Eccedi Importo: OFF tasto verde per salvare e passare alla voce successiva
  • Standalone: ECR tasto verde per salvare e passare alla voce successiva
  • Protocollo V2: ON tasto verde per salvare e passare alla voce successiva
  • Porta SSA: 2638 tasto verde per salvare e passare alla voce successiva

premere tasto rosso per uscire applicazione

8. Release CB2 e Release Applicazioni

8.1.

Release CB2

Per visualizzare la release CB2:

  1. premere tasto F1;
  2. selezionare SERVIZIO;
  3. selezionare RELEASE SW: la seconda linea mostra la release CB2 che è caricata a bordo del pos.
