Pubblicazione su GitHub Pages

1. Carica il contenuto di questo archivio in un repository GitHub.
2. In Settings -> Pages, seleziona il branch desiderato e la cartella root (/).
3. Apri index.html dalla root del sito.

Note:
- La struttura delle cartelle dei PDF è stata mantenuta.
- Ogni cartella contiene un index.html di navigazione.
- I PDF originali non sono inclusi in questa versione web pulita.
